Условие

По нажатию на кнопку создайте шахматную доску:

  • Найдите по id элемент #board
  • Установите position и размер для grid с помощью JavaScript
  • Добавьте в #board 8х8 ячеек (элементы div).
  • Установите для ячеек фон, позицию и размер с помощью JavaScript
  • Запрещается использовать CSS файлы или тег style
  • Для подсказок можете подглядывать в snake\gamefield.js
