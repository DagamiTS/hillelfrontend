My first request

Условие

По нажатию на кнопку:

  • Создайте GET XMLHttpRequest
  • Отправьте запрос используя адрес https://hilleljs.herokuapp.com/hw/42
  • Получите ответ и выведите его в элемент #result
  • Ответ приходит не мгновенно, а со случайной задержкой до 3 секунд.

  • Ссылки на документацию по XMLHttpRequest и disable button

  • Дополнительно. Чтобы предотвратить отправку нескольких запросов одновременно - отключите кнопку при отправке запроса и включите ее вновь, когда ответ получен.
  • Также, в файле server.js можете посмотреть на код, который используется на сервере для отправки ответа на ваши запросы. Его можно запустить и локально используя node.js.
